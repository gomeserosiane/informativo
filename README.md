# Landing Page Grupo Blue & Sabemi

Projeto em HTML, CSS e JavaScript puro, organizado para deploy inicial na Vercel.

## Estrutura

- `index.html`: estrutura da landing page
- `css/`: estilos separados por área
- `js/data.js`: conteúdo dinâmico dos cards e FAQs
- `js/app.js`: navegação, FAQ, animações e WhatsApp

## Deploy na Vercel

Basta importar o repositório/projeto na Vercel como projeto estático.
